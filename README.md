# 🌐 Personal Portfolio Website

A modern, fully responsive, and animated personal portfolio website built using **React**, **Vite**, and **Tailwind CSS**. This portfolio showcases my technical skills, projects, education, and contact information while providing an engaging user experience through smooth animations, responsive layouts, and modern UI design.

The website is designed with performance, accessibility, responsiveness, and clean architecture in mind. It serves as my professional online presence where recruiters, developers, and collaborators can learn about me and explore my work.

# 📖 Project Description

This project is a fully responsive personal portfolio website created from scratch without using any ready-made templates. The primary objective of this project is to present my technical skills, academic background, and software development projects in a clean, modern, and interactive way.

The portfolio demonstrates modern frontend development practices by combining responsive layouts, reusable React components, smooth scrolling animations, optimized assets, and performance-focused development techniques.

The website is designed to work seamlessly across desktop, tablet, and mobile devices while maintaining accessibility standards and fast loading performance.

# ⚡ Animations

To improve user experience, several animations are implemented.

These include:

### Hero Animation

Content smoothly fades into view when the page loads.

### Scroll Reveal Animations

Each section appears with animation while scrolling.

### Project Card Animation

Project cards animate into view with staggered effects.

### Skills Animation

Skill cards smoothly appear during scrolling.

### Hover Effects

Interactive buttons and cards respond with scaling and transition effects.

### Parallax Effect

Background decorative elements move at different speeds to create depth.

# ♿ Accessibility

Accessibility improvements include:

- Semantic HTML
- Keyboard Navigation
- Proper Heading Structure
- Alt Text for Images
- High Color Contrast
- Reduced Motion Support using

```
prefers-reduced-motion
```

This disables animations for users who prefer minimal motion.

# 🖥 Installation

Clone the repository

```
git clone https://github.com/yourusername/portfolio.git
```

Go inside the project folder

```
cd portfolio
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open your browser

```
http://localhost:5173
```

---

# 📦 Build for Production

Generate the production build

```
npm run build
```

Preview production build

```
npm run preview
```
